# Tutorial de Programación Orientada a Objetos en C#

Este repositorio contiene un **tutorial introductorio de Programación Orientada a Objetos (POO)** utilizando el lenguaje **C#**.  

El objetivo del proyecto es explicar los conceptos fundamentales de la POO mediante ejemplos prácticos y un pequeño sistema desarrollado en C#.

---

# Contenido del tutorial

1. Introducción  
2. Clases  
3. Objetos  
4. Atributo y Métodos 
5. Constructores  
6. Encapsulamiento
7. Relaciones entre clases  
8. Ejemplo completo: Sistema de biblioteca  

Cada sección del tutorial se encuentra en archivos **Markdown** dentro de la carpeta `tutorial`.

---

El repositorio incluye ejemplos prácticos de conceptos de **Programación Orientada a Objetos**, tales como:

- Definición de clases
- Creación de objetos
- Encapsulamiento
- Herencia
- Relaciones entre clases
- Uso de propiedades `get` y `set`
- Uso de la palabra reservada `this`
